# 신입 클라이언트 개발자 채용 과제

## 1. 과제 목적

안녕하세요.

버거몬스터 신입 클라이언트 개발자 채용 과제에 참여해 주셔서 감사합니다.

Unity를 활용하여 실제로 플레이 가능한 수준의 **턴제 카드 배틀 게임 프로토타입**을 제작해 주세요.

본 과제의 목적은 완성도 높은 상용 게임을 만드는 것이 아니라, 아래 역량을 평가하는 데 있습니다.

- 게임 시스템 설계 능력
- Unity 클라이언트 개발 역량
- 코드 구조화 설계 능력
- UI/UX 구현 능력
- 게임을 재미있게 만드는 감각
- AI 도구를 활용한 제작 효율성

본 과제는 최근 게임 제작 트렌드에 맞게 **AI 도구를 적극적으로 활용하는 것을 권장합니다.**

코드 작성, 리팩토링, 디버깅, 아트 리소스 제작, UI 시안 제작, 연출 아이디어 구상 등 개발 과정 전반에서 AI를 활용해도 됩니다.

다만 AI 활용 여부와 관계없이, 최종 결과물의 완성도와 재미, 그리고 본인이 구현한 내용을 명확히 설명할 수 있는지를 중점적으로 평가합니다.

## 3. 게임 설명

### 게임 개요

플레이어와 상대방이 번갈아 턴을 진행하는 세로형 턴제 카드 배틀 게임입니다.

각 플레이어는 여러 장의 카드를 보유하며, 전장에 배치된 카드를 사용해 상대 카드를 공격합니다.

상대방의 모든 카드를 제거하면 승리합니다.

### 카드 배치 규칙

게임 시작 시 플레이어와 상대방은 각각 6장의 카드를 보유합니다.

3장의 카드는 전장에 배치되어 공개 상태가 되며, 나머지 카드는 뒤집힌 상태의 대기 카드로 존재합니다.

전장에 빈 슬롯이 발생하고 아직 공개되지 않은 대기 카드가 남아 있다면, 자동으로 새로운 카드가 전장에 배치됩니다.

### 턴 진행 규칙

자신의 턴에는 아래 순서로 행동합니다.

1. 사용할 아군 카드를 선택합니다.
2. 공격 또는 스킬을 선택합니다.
3. 공격 대상 또는 스킬 대상을 선택합니다.
4. 효과를 적용합니다.
5. 턴을 종료합니다.

상대방 턴은 자동 진행 또는 간단한 AI 로직으로 구현해 주세요.

### 체력 규칙

모든 카드는 체력, 즉 HP를 가집니다.

- 카드의 HP가 0 이하가 되면 해당 카드는 제거됩니다.
- 카드가 제거된 후 대기 카드가 남아 있다면 새로운 카드가 전장에 자동 배치됩니다.

### 승리 조건

상대방의 모든 카드가 제거되면 승리합니다.

플레이어의 모든 카드가 제거되면 패배합니다.

### 카드 종류

| 종류 | 설명 |
|------|------|
| 일반 | 자신의 현재 HP만큼 선택한 상대 카드에게 피해를 줍니다. 공격한 카드는 상대 카드의 현재 HP만큼 반격 피해를 받습니다. |
| 원거리 | 자신의 현재 HP만큼 선택한 상대 카드에게 피해를 줍니다. 공격한 카드는 반격 피해를 받지 않습니다. |
| 무쌍 | 선택한 적 카드에게 자신의 현재 HP의 100% 피해를 줍니다. 추가로 선택한 카드와 인접한 적 카드 중 무작위 1장에게 자신의 현재 HP의 50% 피해를 줍니다. 인접한 적 카드가 없다면 추가 피해는 발생하지 않습니다. |
| 힐러 | 자신의 턴 시작 시, 자신을 제외한 아군 카드들의 HP를 1 회복시킵니다. 공격은 일반 카드와 동일하게 동작합니다. |
